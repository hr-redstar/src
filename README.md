# 送迎Bot（送迎者bot）

Discordサーバー内で **送迎依頼・マッチング・相乗り・口コミ評価** を一元管理できる送迎支援Botです。

店舗送迎、イベント送迎、スタッフ送迎など、
「誰が・いつ・どこから・どこへ」送迎するのかを分かりやすく、安全に運用できます。

---

## 主な機能

### 🚗 利用者向け

* 送迎依頼（自分・ゲスト）
* プライベートVCでの送迎進行管理
* 相乗り参加
* 送迎完了後の口コミ評価

### 🧑‍✈️ 送迎者（ドライバー）向け

* 出勤・退勤・休憩管理
* 待機場所の更新
* マッチング対応
* 相乗り受け入れ

### 🛠 管理者向け

* 各種パネルの設置
* ロール・カテゴリー・ログ設定
* 運用状況の可視化
* 評価ログの管理

---

## 基本的な利用フロー

1. **管理者がパネルを設置**
2. **利用者・送迎者が登録**
3. **送迎者が出勤／利用者が依頼**
4. **マッチング → プライベートVC作成**
5. **送迎実施（相乗り対応可）**
6. **送迎完了 → 口コミ評価**

## 特徴

* コマンド入力不要（すべてボタン操作）
* 状態に応じたUI切り替え
* プライベートVCによる安全なやり取り
* 相乗り・定員管理対応
* Ephemeral（本人のみ表示）による操作通知

---

## 想定利用シーン

* 深夜送迎・退勤サポート
