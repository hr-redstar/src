## 構成要素
| 要素 | 内容 | 例 |
| :--- | :--- | :--- |
| **namespace** | 機能カテゴリ | `ps`, `ride`, `reg` |
| **action** | 実行アクション | `send`, `select`, `start`, `end` |
| **params** | 可変パラメータ | `key=value&key2=value2` |

## 命名・運用ルール
- **action**: 動詞・現在形で定義する（send / select / start / cancel 等）。snake_case は避ける。
- **params**: 
    - `key=value` 形式で `&` 連結。
    - 予約キー: `panel` (パネル種別), `vc` (VC ID), `uid` (User ID), `mid` (Message ID)。
- **無効なID**: 仕様に合致しない（パースに失敗する）IDはエラーを投げず、単に無視する。

## 実例
- **パネル設置ボタン**: `ps|send|panel=rideList`
- **チャンネル選択メンバ**: `ps|select|panel=rideList`
- **送迎開始操作**: `ride|start|vc=1234567890`

## 移行期間の対応
過渡期は旧形式（`:` 区切り）も考慮したパーサーを使用し、順次新形式へ移行する。
将来的に `|` 区切りに完全統一する。
